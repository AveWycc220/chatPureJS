## Chat on PureJS / WebSocket with Webpack

#### Installation Packages

`npm i`


#### Commands

Build application. (Use 'production' mode in webpack.config)
`npm run build`

Run dev-server. (Use 'development' mode in webpack.config)
`npm run start`
